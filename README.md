# CodeAlpha Music Player

A simple Music Player built using HTML, CSS, and JavaScript.

## Features

- Play music
- Pause music
- Audio controls
- User-friendly interface
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

CodeAlpha_MusicPlayer/
│
├── index.html
├── style.css
├── script.js
├── song.mp3
└── README.md

## How to Run

1. Download or clone the repository.
2. Place your music file as song.mp3.
3. Open index.html in your browser.
4. Enjoy listening to music.

## Future Improvements

- Next and Previous buttons
- Playlist support
- Progress bar
- Volume control
- Auto-play functionality
